<h2>Project Info: Neon DS</h2>

<p><strong>Neon DS</strong> is a fullstack showcase web application developed for a neon design company. 
It highlights available services and provides users with a modern interface to explore design options.</p>

<h3>Tech Stack</h3>
<ul>
  <li><strong>Frontend:</strong> React (JavaScript)</li>
  <li><strong>Backend:</strong> Node.js</li>
  <li><strong>Architecture:</strong> Full-stack web application</li>
</ul>

<h3>Key Features</h3>
<ul>
  <li>Dynamic and responsive UI built with React</li>
  <li>Showcase of neon design services</li>
  <li>Modular and scalable project structure</li>
  <li>Ready for future integrations (contact forms, database, APIs)</li>
</ul>

<h3>Services Offered</h3>
<ul>
  <li>Custom neon sign boards</li>
  <li>LED lighting solutions</li>
  <li>Decorative neon installations</li>
</ul>

<h3>Purpose</h3>
<ul>
  <li>Demonstrates services offered by the company</li>
  <li>Acts as a portfolio-style showcase website</li>
  <li>Provides users with a clear overview of neon design solutions</li>
</ul>

<h3>Future Improvements</h3>
<ul>
  <li>Add contact form with backend integration</li>
  <li>Implement database for storing customer inquiries</li>
  <li>Enhance UI/UX with animations</li>
</ul>
